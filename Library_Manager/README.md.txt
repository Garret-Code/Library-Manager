A simple Python console application to manage a library of books.  
You can view, add, search, filter, and remove books. The program also prevents duplicate entries.

## Features

- View all books in the library
- Add new books (duplicate prevention)
- Search for a book by title (case-insensitive)
- Filter books by category
- Remove a book by title
- Simple console-based menu for easy navigation
